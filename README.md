# ⚡ Electric Vehicle Data Analysis and Prediction

## 📌 Overview
This project analyzes an electric vehicle dataset using Python to understand performance trends and predict vehicle range.

## 🛠️ Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib / Seaborn (for visualization)
- Scikit-learn (for regression model)

## 🔍 Analysis Performed
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature relationship analysis  
- Data visualization for trend identification  

## 📈 Key Insights
- Identified key factors affecting vehicle range  
- Observed trends in vehicle performance  
- Highlighted relationships between variables  

## 🤖 Model
- Built a regression model to predict EV range  
- Evaluated model performance using standard metrics  

## ▶️ How to Use
1. Download the dataset  
2. Open the Jupyter Notebook (`.ipynb`)  
3. Run all cells to reproduce results
